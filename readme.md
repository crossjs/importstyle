 > import style text with js, seajs module

## how to demo

1. checkout
1. run `npm install`
1. run `spm install`
1. run `grunt`
1. view files in `/demo`

## how to use

1. run `spm install pandora/importstyle`
1. write `require('pandora/importstyle/VERSION.NUMBER/importstyle')`

## find examples

1. view the source files in '/src'
